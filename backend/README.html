Projekt Áttekintés

Ez a projekt egy modern, RESTful API backend alkalmazás energiaitalok kezelésére. Az alkalmazás Express.js keretrendszert használ Node.js környezetben, MySQL adatbázissal a háttérben.

API Documentation

Base URL: http://localhost:3000

GET /energy_drinks

Összes energiaital lekérdezése a rendszerből

[ { "id": 1, "name": "Red Bull Energy", "brand": "Red Bull GmbH", "caffeine": 80, "sugar_free": false, "created_at": "2025-10-02T10:00:00.000Z", "updated_at": "2025-10-02T10:00:00.000Z" } ]

POST /energy_drinks

Új energiaital hozzáadása az adatbázishoz

{ "name": "Monster Energy Ultra", "brand": "Monster Beverage Corporation", "caffeine": 160, "sugar_free": true }

PUT /energy_drinks/:id

Meglévő energiaital adatainak módosítása

DELETE /energy_drinks/:id

Energiaital törlése az adatbázisból

Quick Start Guide

Repository klónozása

git clone <repository-url> cd backend

Függőségek telepítése

npm install

Adatbázis beállítása

CREATE DATABASE energy_drinks; USE energy_drinks; CREATE TABLE drinks ( id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(255) NOT NULL, brand VARCHAR(255) NOT NULL, caffeine INT NOT NULL, sugar_free BOOLEAN NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP );

Szerver indítása

npm start

A szerver elindul a http://localhost:3000 címen
